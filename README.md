# Bulgaria Travel Guide

## Description

This is a web application for travelers that provides information about beautiful destinations in Bulgaria. It allows user registration and login, adding comments in a forum, and viewing details for each destination.  

## Features

- View top destinations and all destinations in Bulgaria.  
- User registration and login.  
- Authentication and page protection using AuthGuard.  
- Forum for sharing opinions and comments with the ability to add, edit, and delete comments (CRUD).  
- Reactive interface using React, Context API, and hooks.

- React.js

- Fetch API for server communication  
- CSS Modules for styling  

## Running the Application

1. Clone the repository to your local machine.  
2. Start the server: `node server.js` (SoftUni training server).  
3. Start the frontend application: `npm run dev`.  
4. Open in your browser at: `http://localhost:5173/`.
